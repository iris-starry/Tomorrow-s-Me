# Tomorrow's Me
2022 JAVA 프로젝트 '내일의 나' 교훈일기

## 🖥️ 프로젝트 소개
내가 느낀 명언, 다짐을 더 나은 내일을 위해 간단명료하게 내가 나의 인생 조언을 기록할 수 있는 교훈일기

<br/>

## 🕰️ 개발 기간
* 22.08.27일 - 22.10.28일

### ⚙️ 개발 환경
- `Java 17`
- `JDK 17.0.8`
- **IDE** : Eclipse
- **Database** : MySQL 8.0.33
